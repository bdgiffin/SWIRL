SWIRL is a Structural Wind-borne debris Impact Risk assessment Library developed and intended for research purposes. SWIRL was designed to be coupled with the open source structural analysis software OpenSEES.

SWIRL is written in C++ and Python, with corresponding language-specific APIs.

Note

The documentation for SWIRL is under active development. Please check back periodically for updates.
